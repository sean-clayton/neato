## Features

### Introduction

Neato is filled with a few features, but most are hidden unless you know they're there. This documentation is made to solve that for you :)

### Feature List

- [Processors]
  - ES2015+ JavaScript
  - TypeScript (2.x supported)
  - PostCSS
  - Images
- [Helpers]
  - Environment Globals
  - Offline Support
  - Module Resolution
